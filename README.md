# Calculadora de Vectores y Matrices

Esta es una aplicación de escritorio que permite realizar operaciones básicas con vectores y matrices, como sumar, restar, calcular el producto punto, la magnitud, la proyección, el producto matriz-vector y el producto de matrices. La aplicación utiliza la biblioteca Tkinter para la interfaz gráfica y NumPy para los cálculos matemáticos.

## Características

- Interfaz gráfica intuitiva y fácil de usar.
- Cálculo de operaciones con vectores y matrices.
- Visualización gráfica de los vectores y matrices.
- Soporte para matrices de 3x3.

## Requisitos

- Python 3.x
- Tkinter
- NumPy
- Matplotlib

## Instalación

1. Clona el repositorio:
``` python
https://github.com/CARLOSGRCIAGRCIA/Collaborativework.git
```

2. Crea y activa un entorno virtual:
```python
cd calculadora-vectores-matrices

python -m venv venv

source venv/bin/activate  # En Windows: venv\Scripts\activate

```

3. Instala las dependencias:

``` python 
pip install -r requirements.txt
```
4. Ejecuta la aplicación:

``` python
python main.py
```

## Uso

1. Ingresa los valores de los vectores o las matrices en los campos correspondientes.
2. Selecciona la operación que deseas realizar en el menú desplegable.
3. Haz clic en el botón "Calcular" para obtener el resultado.
4. La aplicación mostrará el resultado y, en algunos casos, generará gráficos 3D de los vectores o matrices.
